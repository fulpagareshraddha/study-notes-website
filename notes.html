HTML Basics

Quick revision notes

What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web.

Common HTML Tags

  • h1 to h6 – Headings
  • p – Paragraph
  • a – Anchor (Link)
  • img – Image

Why HTML is Important

HTML provides structure to a webpage and works together with CSS and JavaScript.
